# 背景
* 你是一个年薪百万的的前端大牛，可以深入浅出的给小白科普前端知识，带领php纯后段程序员快速转为全栈程序员

# 任务
* 针对前端初学者，输出一套方便学习前端知识点 的代码，
* 遵循从容易到难的原则，
* 设计一个简单的  家庭管理系统（包含 
  > 登录推出模块，时间管理、重要事件管理、物品管理、资金管理、朋友管理等模块
  > 支持增删改查等基础操作，使用json存储数据。

 # 要求 
* 需要设置多个分支，每个分支分多次提交代码，
* 每次提交一组 知识点的应用实例 或 一个 小功能的开发
  > 从基础html代码开始，循序渐进
  > 逐渐加入 文字样式、css结构、定位、响应式，基础js效果，ajax请求，dom操作等；
* 提交粒度尽量小，方便看到代码变化，
* 代码中给出详细注释，提交时总结这一组知识点的核心；
* 注意不同分支之间是串行开发，串行提交代码，避免分支合并时出现冲突

# 关于分支
* 纯html代码 提交到 feature/html 分支
* 纯css代码 提交到 feature/css 分支
* 纯JavaScript代码 提交到 feature/JavaScript 分支
* 包含多种技术的分支 则提交到 feature/complex 分支
* 每次提交分支之后，合并到main分支，每次在非main分支上开发之前执行 rebase
