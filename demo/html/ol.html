有序列表标签 (ol)

基本用法

ol标签用于创建有序列表，列表项使用li标签表示，默认使用数字编号。

  1. 准备HTML结构
  2. 添加CSS样式
  3. 编写JavaScript交互
  4. 测试和调试

代码示例：

<ol> <li>准备HTML结构</li> <li>添加CSS样式</li> <li>编写JavaScript交互</li> <li>测试和调试</li> </ol>

列表样式类型

可以通过CSS的list-style-type属性改变列表项的编号样式：

默认样式（decimal）

  1. 默认数字编号
  2. 顺序排列

小写字母（lower-alpha）

  1. 小写字母编号
  2. 顺序排列

大写字母（upper-alpha）

  1. 大写字母编号
  2. 顺序排列

小写罗马数字（lower-roman）

  1. 小写罗马数字编号
  2. 顺序排列

大写罗马数字（upper-roman）

  1. 大写罗马数字编号
  2. 顺序排列

前导零数字（decimal-leading-zero）

  1. 前导零数字编号
  2. 顺序排列

代码示例：

<!-- 小写字母 --> <ol class="lower-alpha"> <li>小写字母编号</li> <li>顺序排列</li> </ol> <!-- 小写罗马数字 --> <ol class="lower-roman"> <li>小写罗马数字编号</li> <li>顺序排列</li> </ol> <style> .lower-alpha { list-style-type: lower-alpha; } .lower-roman { list-style-type: lower-roman; } </style>

自定义起始编号

可以使用start属性指定有序列表的起始编号：

  1. 第五步
  2. 第六步
  3. 第七步

代码示例：

<ol start="5"> <li>第五步</li> <li>第六步</li> <li>第七步</li> </ol>

反向排序

可以使用reversed属性实现反向排序的有序列表：

  1. 最后一步
  2. 倒数第二步
  3. 倒数第三步

代码示例：

<ol reversed> <li>最后一步</li> <li>倒数第二步</li> <li>倒数第三步</li> </ol>

自定义列表项编号

可以使用value属性为特定列表项指定自定义编号：

  1. 第一步
  2. 第二步
  3. 第十步
  4. 第十一步
  5. 第十二步

代码示例：

<ol> <li>第一步</li> <li>第二步</li> <li value="10">第十步</li> <li>第十一步</li> <li>第十二步</li> </ol>

嵌套列表

有序列表可以与其他列表嵌套使用，创建复杂的层级结构：

  1. 前端开发
    • HTML
    • CSS
    • JavaScript
  2. 后端开发
    1. PHP
    2. Python
    3. Node.js
      1. Express
      2. Koa
      3. NestJS
  3. 数据库
    • MySQL
    • MongoDB

代码示例：

<ol> <li>前端开发 <ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul> </li> <li>后端开发 <ol class="lower-alpha"> <li>PHP</li> <li>Python</li> <li>Node.js <ol class="lower-roman"> <li>Express</li> <li>Koa</li> <li>NestJS</li> </ol> </li> </ol> </li> </ol>
