HTML 定义描述标签 (dd)

<dd> 标签用于在定义列表中提供术语（<dt>）的描述或定义，与 <dl> 和 <dt> 标签配合使用。

它用于解释或描述前面的 <dt> 元素中定义的术语，通常会被浏览器自动缩进显示。

基本语法

<dl>
    <dt>术语</dt>
    <dd>术语的定义或描述</dd>
</dl>

使用示例

示例1：基本使用

代码：

<dl>
    <dt>HTTP</dt>
    <dd>超文本传输协议，用于在Web浏览器和服务器之间传输数据。</dd>
    
    <dt>HTTPS</dt>
    <dd>安全的HTTP协议，通过SSL/TLS加密数据传输。</dd>
    
    <dt>URL</dt>
    <dd>统一资源定位符，用于标识互联网上的资源。</dd>
</dl>

效果：

HTTP
超文本传输协议，用于在Web浏览器和服务器之间传输数据。
HTTPS
安全的HTTP协议，通过SSL/TLS加密数据传输。
URL
统一资源定位符，用于标识互联网上的资源。

<dl>

    <dt class="custom-dt">SQL</dt>
    <dd class="custom-dd">结构化查询语言，用于与数据库进行交互和管理数据。</dd>
    
    <dt class="custom-dt">NoSQL</dt>
    <dd class="custom-dd">非关系型数据库，用于处理大量非结构化或半结构化数据。</dd>
</dl>

效果：

数据库
存储和管理数据的系统，用于高效地检索、更新和管理数据。
SQL
结构化查询语言，用于与数据库进行交互和管理数据。

非关系型数据库，用于处理大量非结构化或半结构化数据。

代码：

        </dl>
    </dd>
</dl>

效果：

服务器

Web服务器
处理HTTP请求并返回网页内容。

使用注意事项

  • 语义化意义：<dd> 标签具有明确的语义，用于提供术语的定义或描述，有助于搜索引擎和辅助技术理解内容结构。
  • 正确嵌套：<dd> 只能作为 <dl> 标签的直接子元素，且通常紧跟在 <dt> 元素之后。
  • 多个描述：一个 <dt> 元素可以对应多个 <dd> 元素，表示一个术语有多个定义或描述。
  • 默认样式：浏览器通常会对 <dd> 元素应用左缩进样式，这有助于视觉上区分术语和描述。
  • 自定义样式：可以使用CSS自定义 <dd> 元素的外观，包括缩进、字体、颜色、背景等。
  • 内容丰富性：<dd> 元素内可以包含各种HTML内容，如文本、链接、图片、列表等。

浏览器支持

所有现代浏览器都支持 <dd> 标签：

  • Chrome (所有版本)
  • Firefox (所有版本)
  • Safari (所有版本)
  • Edge (所有版本)
  • Opera (所有版本)

与其他标签的关系

<dd> 标签通常与以下标签一起使用：

  • <dl>：定义列表容器
  • <dt>：定义列表中的术语

这三个标签共同构成了完整的定义列表结构，用于展示术语-定义的对应关系。
