列表项标签 (li)

基本用法

li标签用于定义列表中的项目，必须嵌套在ul（无序列表）或ol（有序列表）中。

在无序列表中使用

  • HTML
  • CSS
  • JavaScript

在有序列表中使用

  1. 准备HTML结构
  2. 添加CSS样式
  3. 编写JavaScript交互

代码示例：

<!-- 在无序列表中 --> <ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul> <!-- 在有序列表中 --> <ol> <li>准备HTML结构</li> <li>添加CSS样式</li> <li>编写JavaScript交互</li> </ol>

嵌套列表中的列表项

列表项可以包含嵌套列表，创建层级结构：

  • 前端开发
    • HTML
    • CSS
      • Flexbox
      • Grid
    • JavaScript
  • 后端开发
    • PHP
    • Python

代码示例：

<ul> <li>前端开发 <ul> <li>HTML</li> <li>CSS <ul> <li>Flexbox</li> <li>Grid</li> </ul> </li> <li>JavaScript</li> </ul> </li> <li>后端开发 <ul> <li>PHP</li> <li>Python</li> </ul> </li> </ul>

列表项的value属性

在有序列表中，可以使用value属性为列表项指定自定义编号：

  1. 第一步
  2. 第五步
  3. 第六步
  4. 第十步

代码示例：

<ol> <li>第一步</li> <li value="5">第五步</li> <li>第六步</li> <li value="10">第十步</li> </ol>
