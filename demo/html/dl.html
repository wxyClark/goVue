定义列表标签 (dl)

基本用法

dl标签用于创建定义列表，包含定义术语(dt)和定义描述(dd)。

定义列表常用于展示术语和解释、问答等配对信息。

HTML
超文本标记语言，用于创建网页结构。
CSS
层叠样式表，用于设计网页外观。
JavaScript
脚本语言，用于实现网页交互功能。

代码示例：

<dl> <dt>HTML</dt> <dd>超文本标记语言，用于创建网页结构。</dd> <dt>CSS</dt> <dd>层叠样式表，用于设计网页外观。</dd> <dt>JavaScript</dt> <dd>脚本语言，用于实现网页交互功能。</dd> </dl>

自定义样式的定义列表

可以通过CSS为定义列表添加自定义样式：

前端开发
负责构建用户可以直接看到和交互的网页部分。
后端开发
负责处理服务器端逻辑、数据库交互和API开发。
全栈开发
同时掌握前端和后端开发技能的开发者。

代码示例：

<dl class="custom-dl"> <dt>前端开发</dt> <dd>负责构建用户可以直接看到和交互的网页部分。</dd> <dt>后端开发</dt> <dd>负责处理服务器端逻辑、数据库交互和API开发。</dd> </dl> <style> .custom-dl dt { font-weight: bold; color: #4CAF50; margin-top: 1rem; } .custom-dl dd { margin-left: 1rem; color: #666; } </style>

一个术语多个描述

一个定义术语(dt)可以对应多个定义描述(dd)：

Web开发
前端开发：构建用户界面
后端开发：处理服务器逻辑
全栈开发：同时掌握前后端技能

代码示例：

<dl> <dt>Web开发</dt> <dd>前端开发：构建用户界面</dd> <dd>后端开发：处理服务器逻辑</dd> <dd>全栈开发：同时掌握前后端技能</dd> </dl>

复杂内容的定义列表

定义列表中的dt和dd可以包含各种HTML内容：

响应式设计

响应式设计是一种网页设计方法，使网页能够适应不同屏幕尺寸和设备。

  • 使用CSS媒体查询
  • 弹性布局和网格系统
  • 流体图像和媒体
语义化HTML

语义化HTML是指使用具有明确含义的HTML标签，使网页内容更易于理解。

  • <header>：页面头部
  • <nav>：导航菜单
  • <main>：主要内容
  • <footer>：页面底部

代码示例：

<dl> <dt> <strong>响应式设计</strong> </dt> <dd> <p>响应式设计是一种网页设计方法，使网页能够适应不同屏幕尺寸和设备。</p> <ul> <li>使用CSS媒体查询</li> <li>弹性布局和网格系统</li> <li>流体图像和媒体</li> </ul> </dd> <dt> <strong>语义化HTML</strong> </dt> <dd> <p>语义化HTML是指使用具有明确含义的HTML标签，使网页内容更易于理解。</p> <ul> <li>&lt;header&gt;：页面头部</li> <li>&lt;nav&gt;：导航菜单</li> <li>&lt;main&gt;：主要内容</li> </ul> </dd> </dl>

定义列表的使用场景

定义列表适用于以下场景：

  • 术语表和词汇解释
  • 产品特性说明
  • FAQ（常见问题解答）
  • 属性和值的配对展示
  • API文档
