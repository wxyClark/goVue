无序列表标签 (ul)

基本用法

ul标签用于创建无序列表，列表项使用li标签表示。

  • HTML
  • CSS
  • JavaScript
  • React

代码示例：

<ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> <li>React</li> </ul>

列表样式类型

可以通过CSS的list-style-type属性改变列表项的标记样式：

默认样式（disc）

  • 默认样式
  • 圆形标记

方形样式（square）

  • 方形样式
  • 方形标记

空心圆样式（circle）

  • 空心圆样式
  • 空心圆标记

无标记样式（none）

  • 无标记样式
  • 不显示标记

代码示例：

<!-- 默认样式 --> <ul> <li>默认样式</li> <li>圆形标记</li> </ul> <!-- 方形样式 --> <ul class="custom-list"> <li>方形样式</li> <li>方形标记</li> </ul> <style> .custom-list { list-style-type: square; } </style>

自定义标记图像

可以通过CSS的list-style-image属性使用自定义图像作为列表标记：

  • 自定义图像标记
  • 绿色圆形标记

代码示例：

<ul class="image-list"> <li>自定义图像标记</li> <li>绿色圆形标记</li> </ul> <style> .image-list { list-style-image: url('data:image/svg+xml;utf8,<svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 16 16"><circle cx="8" cy="8" r="6" fill="%234CAF50"/></svg>'); } </style>

列表项位置

可以通过CSS的list-style-position属性控制列表标记的位置：

默认位置（outside）

  • 这是一个很长的列表项，用于演示默认的标记位置。可以看到标记在列表项内容的左侧外部。
  • 这是第二个列表项。

内部位置（inside）

  • 这是一个很长的列表项，用于演示内部的标记位置。可以看到标记在列表项内容的左侧内部。
  • 这是第二个列表项。

代码示例：

<!-- 默认位置 --> <ul> <li>这是一个很长的列表项...</li> <li>这是第二个列表项。</li> </ul> <!-- 内部位置 --> <ul class="position-list"> <li>这是一个很长的列表项...</li> <li>这是第二个列表项。</li> </ul> <style> .position-list { list-style-position: inside; } </style>
