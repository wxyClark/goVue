运算符

JavaScript提供了各种运算符，用于执行算术运算、比较、逻辑运算等操作。

算术运算符

运算符 描述 示例 结果
+ 加法 5 + 3 8
- 减法 5 - 3 2
* 乘法 5 * 3 15
/ 除法 5 / 3 1.666...
% 取模（余数） 5 % 3 2
++ 自增 let x = 5; x++ x变为6
-- 自减 let x = 5; x-- x变为4
let a = 10; let b = 3; console.log(a + b); // 13 console.log(a - b); // 7 console.log(a * b); // 30 console.log(a / b); // 3.3333333333333335 console.log(a % b); // 1 // 自增和自减 let x = 5; console.log(x++); // 5 (先返回x，再自增) console.log(x); // 6 let y = 5; console.log(++y); // 6 (先自增，再返回y) console.log(y); // 6

赋值运算符

运算符 描述 示例 等价于
= 赋值 x = 5 x = 5
+= 加法赋值 x += 3 x = x + 3
-= 减法赋值 x -= 3 x = x - 3
*= 乘法赋值 x *= 3 x = x * 3
/= 除法赋值 x /= 3 x = x / 3
%= 取模赋值 x %= 3 x = x % 3
let x = 5; x += 3; // x = x + 3 console.log(x); // 8 x -= 2; // x = x - 2 console.log(x); // 6 x *= 4; // x = x * 4 console.log(x); // 24 x /= 3; // x = x / 3 console.log(x); // 8 x %= 5; // x = x % 5 console.log(x); // 3
