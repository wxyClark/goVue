控制流

JavaScript的控制流语句用于根据条件执行不同的代码块，或者重复执行特定的代码块。

条件语句

1. if语句

if语句用于根据条件执行代码块。

let age = 18; if (age >= 18) { console.log("你是成年人"); }
输出：你是成年人

2. if...else语句

if...else语句用于在条件为真时执行一个代码块，为假时执行另一个代码块。

let score = 75; if (score >= 60) { console.log("考试通过"); } else { console.log("考试不及格"); }
输出：考试通过

3. if...else if...else语句

if...else if...else语句用于检查多个条件。

let score = 85; if (score >= 90) { console.log("等级：A"); } else if (score >= 80) { console.log("等级：B"); } else if (score >= 70) { console.log("等级：C"); } else if (score >= 60) { console.log("等级：D"); } else { console.log("等级：F"); }
输出：等级：B

4. switch语句

switch语句用于根据不同的情况执行不同的代码块。

let day = 3; let dayName; switch (day) { case 1: dayName = "星期一"; break; case 2: dayName = "星期二"; break; case 3: dayName = "星期三"; break; case 4: dayName = "星期四"; break; case 5: dayName = "星期五"; break; case 6: dayName = "星期六"; break; case 7: dayName = "星期日"; break; default: dayName = "无效的日期"; } console.log(dayName);
输出：星期三
注意：每个case后面必须使用break语句来终止switch语句，否则会继续执行下一个case（称为"穿透"）。

循环语句

1. for循环

for循环用于重复执行代码块指定的次数。

// 语法：for (初始化; 条件; 递增/递减) { // 要执行的代码块 // } for (let i = 0; i < 5; i++) { console.log("循环次数：" + i); }
输出： 循环次数：0 循环次数：1 循环次数：2 循环次数：3 循环次数：4

2. while循环

while循环用于在条件为真时重复执行代码块。

let i = 0; while (i < 5) { console.log("while循环次数：" + i); i++; }
输出： while循环次数：0 while循环次数：1 while循环次数：2 while循环次数：3 while循环次数：4
注意：确保while循环的条件最终会变为假，否则会导致无限循环。

3. do...while循环

do...while循环会先执行一次代码块，然后再检查条件。

let i = 0; do { console.log("do...while循环次数：" + i); i++; } while (i < 5);
输出： do...while循环次数：0 do...while循环次数：1 do...while循环次数：2 do...while循环次数：3 do...while循环次数：4
注意：do...while循环至少会执行一次代码块，即使条件一开始就是假的。

4. for...in循环

for...in循环用于遍历对象的属性。

let person = { name: "张三", age: 25, isStudent: true }; for (let key in person) { console.log(key + ": " + person[key]); }
输出： name: 张三 age: 25 isStudent: true

5. for...of循环

for...of循环用于遍历数组或其他可迭代对象的值。

let fruits = ["苹果", "香蕉", "橙子"]; for (let fruit of fruits) { console.log(fruit); } // 遍历字符串 let str = "Hello"; for (let char of str) { console.log(char); }
输出： 苹果 香蕉 橙子 H e l l o

跳转语句

1. break语句

break语句用于终止循环或switch语句。

for (let i = 0; i < 10; i++) { if (i === 5) { break; // 当i等于5时终止循环 } console.log("i的值：" + i); }
输出： i的值：0 i的值：1 i的值：2 i的值：3 i的值：4

2. continue语句

continue语句用于跳过当前循环的剩余部分，进入下一次循环。

for (let i = 0; i < 10; i++) { if (i % 2 === 0) { continue; // 跳过偶数 } console.log("i的值：" + i); }
输出： i的值：1 i的值：3 i的值：5 i的值：7 i的值：9
