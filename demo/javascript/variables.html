变量和数据类型

变量声明

JavaScript中可以使用三种方式声明变量：var、let和const。

1. 使用var声明变量

var name = "张三"; var age = 25; var isStudent = true; console.log(name, age, isStudent);
输出：张三 25 true
注意：var声明的变量具有函数作用域或全局作用域，可能会导致变量提升问题，现代JavaScript中推荐使用let和const。

2. 使用let声明变量

let name = "李四"; let age = 30; age = 31; // 可以重新赋值 console.log(name, age);
输出：李四 31
注意：let声明的变量具有块级作用域，不会被提升，推荐在大多数情况下使用。

3. 使用const声明常量

const PI = 3.14159; const name = "王五"; // PI = 3.14; // 这会抛出错误，常量不能重新赋值 console.log(PI, name);
输出：3.14159 王五
注意：const声明的变量是常量，具有块级作用域，不能重新赋值，但对象和数组的内容可以修改。

数据类型

JavaScript中有7种基本数据类型和1种引用数据类型：

1. 基本数据类型

// 字符串类型 let str = "Hello World"; // 数字类型 let num1 = 42; let num2 = 3.14; let num3 = NaN; // Not a Number let num4 = Infinity; // 无穷大 // 布尔类型 let isTrue = true; let isFalse = false; // Null类型 let nullValue = null; // Undefined类型 let undefinedValue; // BigInt类型 (ES2020) let bigInt = 123456789012345678901234567890n; // Symbol类型 (ES6) let symbol = Symbol("unique"); console.log(typeof str); // string console.log(typeof num1); // number console.log(typeof isTrue); // boolean console.log(typeof nullValue); // object (这是一个历史遗留问题) console.log(typeof undefinedValue); // undefined console.log(typeof bigInt); // bigint console.log(typeof symbol); // symbol

2. 引用数据类型

// 对象 let person = { name: "张三", age: 25, isStudent: true }; // 数组 let fruits = ["苹果", "香蕉", "橙子"]; // 函数 function greet() { console.log("Hello"); } console.log(typeof person); // object console.log(typeof fruits); // object console.log(typeof greet); // function

类型转换

JavaScript会自动进行类型转换，也可以手动进行类型转换。

1. 自动类型转换

// 数字转字符串 let num = 42; let str = "The answer is " + num; console.log(str); // "The answer is 42" // 字符串转数字 let result = "10" - "5"; console.log(result); // 5 // 布尔值转数字 let total = 100 + true; // true转换为1 console.log(total); // 101 // 数字转布尔值 if (0) { console.log("这不会执行"); } else { console.log("0被转换为false"); }
输出： The answer is 42 5 101 0被转换为false

2. 手动类型转换

// 转换为字符串 let num = 42; let str1 = String(num); let str2 = num.toString(); console.log(str1, str2); // "42" "42" // 转换为数字 let str = "42"; let num1 = Number(str); let num2 = parseInt(str); let num3 = parseFloat("3.14"); console.log(num1, num2, num3); // 42 42 3.14 // 转换为布尔值 let bool1 = Boolean("Hello"); // true let bool2 = Boolean(0); // false let bool3 = Boolean(null); // false console.log(bool1, bool2, bool3); // true false false
输出： 42 42 42 42 3.14 true false false
