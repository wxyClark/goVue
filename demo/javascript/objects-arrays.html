JavaScript对象和数组

对象（Object）

对象是JavaScript中最基本的数据结构，用于存储键值对（属性和方法）。

对象的创建

1. 对象字面量方式

// 对象字面量 const person = { name: "张三", age: 30, gender: "男", sayHello: function() { console.log(`你好，我是${this.name}`); } };

2. Object构造函数方式

// Object构造函数 const person2 = new Object(); person2.name = "李四"; person2.age = 25; person2.sayHello = function() { console.log(`你好，我是${this.name}`); };

3. Object.create()方式

// Object.create() const person3 = Object.create(null); person3.name = "王五"; person3.age = 28;

对象属性的访问

// 点表示法 console.log(person.name); // 输出：张三 // 方括号表示法 console.log(person["age"]); // 输出：30 // 动态属性名 const prop = "gender"; console.log(person[prop]); // 输出：男

对象方法的调用

person.sayHello(); // 输出：你好，我是张三

对象的遍历

1. for...in循环

for (let key in person) { console.log(`${key}: ${person[key]}`); }

2. Object.keys()、Object.values()、Object.entries()

// 获取所有键 const keys = Object.keys(person); console.log(keys); // 输出：["name", "age", "gender", "sayHello"] // 获取所有值 const values = Object.values(person); console.log(values); // 输出：["张三", 30, "男", function] // 获取所有键值对 const entries = Object.entries(person); console.log(entries); // 输出：[["name", "张三"], ["age", 30], ...]

数组（Array）

数组是用于存储多个值的有序集合，可以包含任何类型的数据。

数组的创建

1. 数组字面量方式

// 数组字面量 const fruits = ["苹果", "香蕉", "橙子", "葡萄"]; const mixed = [1, "字符串", true, null, undefined, {name: "对象"}, [1, 2, 3]];

2. Array构造函数方式

// Array构造函数 const numbers = new Array(1, 2, 3, 4, 5); const emptyArray = new Array(10); // 创建长度为10的空数组

数组元素的访问

console.log(fruits[0]); // 输出：苹果 console.log(fruits[2]); // 输出：橙子 console.log(fruits.length); // 输出：4 console.log(fruits[fruits.length - 1]); // 输出：葡萄（最后一个元素）

常用数组方法

方法 描述 示例
push() 在数组末尾添加一个或多个元素 fruits.push("草莓")
pop() 移除数组最后一个元素并返回它 fruits.pop()
shift() 移除数组第一个元素并返回它 fruits.shift()
unshift() 在数组开头添加一个或多个元素 fruits.unshift("西瓜")
slice() 返回数组的一部分，不修改原数组 fruits.slice(1, 3)
splice() 添加/移除数组元素，修改原数组 fruits.splice(1, 1, "芒果")
concat() 合并两个或多个数组，不修改原数组 fruits.concat(vegetables)
join() 将数组元素连接成字符串 fruits.join(", ")
reverse() 颠倒数组元素顺序，修改原数组 fruits.reverse()
sort() 对数组元素进行排序，修改原数组 fruits.sort()

数组遍历方法

1. for循环

for (let i = 0; i < fruits.length; i++) { console.log(fruits[i]); }

2. forEach()

fruits.forEach(function(fruit, index) { console.log(`${index}: ${fruit}`); });

3. map()

// 创建新数组，不修改原数组 const capitalFruits = fruits.map(function(fruit) { return fruit.toUpperCase(); }); console.log(capitalFruits);

4. filter()

// 筛选符合条件的元素 const evenNumbers = numbers.filter(function(num) { return num % 2 === 0; }); console.log(evenNumbers);

5. reduce()

// 累加数组元素 const sum = numbers.reduce(function(total, num) { return total + num; }, 0); console.log(sum);

对象和数组的应用

对象数组

// 学生对象数组 const students = [ { name: "张三", age: 20, score: 90 }, { name: "李四", age: 21, score: 85 }, { name: "王五", age: 19, score: 95 } ]; // 查找分数最高的学生 const topStudent = students.reduce(function(top, student) { return student.score > top.score ? student : top; }, students[0]); console.log("分数最高的学生:", topStudent);
